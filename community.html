Almost there — connect Giscus

This comment board is built and ready. It needs a one-time connection to GitHub Discussions before comments go live. From the repository:

  1. Make sure the repo is public and turn on Settings → Discussions.
  2. Install the Giscus app on the repo: github.com/apps/giscus.
  3. Open giscus.app, enter infrastructureinvestor/infrastructureinvestor, choose a Discussion category (e.g. General), and copy the generated data-repo-id and data-category-id.
  4. Paste those two values into the GISCUS config near the bottom of community.html.

Until then, visitors see this friendly notice instead of an error.
